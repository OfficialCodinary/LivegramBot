# LivegramBot
An eye-to-eye copy of the famous livegram bot

##Setup
It's easy! Just put the below properties in .env file!
 - `bot_token` (required)
 - `dbFile` (required)
 - `ownerID` (required)
 - `prettyDb` (optional)

The descriptions are pretty self explanotary. So that's it?

### Running your bot
There's nothing like rocket science in this
```shell
npm start
or
node main
```
Will do the job
